# Despliegue automático de una aplicación escalable en OpenStack mediante Terraform 
En este proyecto se implementa un entorno en OpenStack orquestado mediante Terraform, siguiendo el enfoque de Infraestructura como Código (IaC), donde la infraestructura se define de forma declarativa.
El objetivo es desplegar una aplicación web escalable compuesta por varios servidores web, un componente de base de datos aislado, un balanceador de carga que distribuya el tráfico entrante y un control de seguridad que restrinja el acceso desde el exterior a lo estrictamente necesario. Todo el escenario se crea y configura automáticamente, incluyendo la inicialización de las máquinas virtuales mediante cloud-init.
